# CONVÊNIO — PRODENTAL BRASIL

## Regra Geral

A cobertura de exames pelo convênio **Prodental Brasil deve sempre ser verificada com a clínica** antes de confirmar qualquer atendimento.

A IX Radiologia **não confirma exames automaticamente** para este convênio.

## Exames

- A cobertura de exames **não é fixa**
- Cada solicitação deve ser analisada individualmente
- Não confirmar realização de nenhum exame sem verificação prévia

## Procedimento Obrigatório

Quando o cliente informar que utiliza o convênio Prodental Brasil:

1. **NÃO confirmar cobertura**
2. **NÃO listar exames aceitos**
3. Coletar:
   - Nome completo  
   - CPF  
   - Data de nascimento  
4. Informar que a solicitação será verificada
5. Transferir para atendente humano conforme protocolo

## Guia

### Tipo de Guia
- **APENAS guia física**
- Guia em papel timbrado é obrigatória
- **Não aceita guia digital**

### Validade da Guia
- **30 dias** a partir da data de emissão

## Observações Importantes

- Guia física original é obrigatória para qualquer verificação
- Atenção à validade curta da guia (30 dias)
- Sem guia física válida, o exame não pode ser realizado
