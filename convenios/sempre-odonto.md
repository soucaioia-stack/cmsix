# CONVÊNIO — SEMPRE ODONTO

Inclui: **CAEDU**

## Regra Geral

Para realizar **qualquer exame** pelo Sempre Odonto, é **OBRIGATÓRIO** possuir o **Pass Card (autorização prévia)** válido.

Sem o Pass Card:
- NÃO confirmar realização de exame
- NÃO orientar ida à clínica
- Orientar a obtenção da autorização

## Exames Aceitos

**TODOS os exames** são aceitos, desde que haja Pass Card válido:
- Documentação Ortodôntica
- Panorâmica
- Tomografia Computadorizada
- Periapical / Interproximal / Bite Wing
- Levantamento Periapical
- Carpal
- ATM
- Telerradiografia
- Moldagem
- Fotos Intraorais/Extraorais
- Escaneamento Intraoral
- Modelo Ortodôntico

## Exames NÃO Aceitos

- Nenhum. Todos os exames são aceitos **com Pass Card válido**.

## Pass Card — Autorização Obrigatória

### O que é o Pass Card?

É uma **autorização obrigatória** exigida pelo Sempre Odonto para realizar **qualquer exame**.

### Validade do Pass Card

- **15 dias** a partir da data de emissão

### Procedimento Obrigatório

Quando o cliente mencionar **Sempre Odonto** ou **CAEDU**:

1. **Perguntar obrigatoriamente**:  
   “Você já tem o Pass Card (autorização)?”

2. Avaliar a resposta:

#### Se NÃO tiver Pass Card

**RESPOSTA LITERAL:**

"Para fazer exames pelo Sempre Odonto, é necessário ter o Pass Card (autorização prévia). O contato para solicitar fica atrás da carteirinha, você pode ligar lá ou pedir para o seu dentista credenciado solicitar. Assim que tiver a autorização, você pode vir fazer o exame aqui. Posso te ajudar com mais alguma coisa?"

❌ Não avançar para informações de exame  
❌ Não confirmar atendimento  

#### Se JÁ tiver Pass Card

- Prosseguir normalmente com informações do exame solicitado

### Como obter o Pass Card

- O contato fica **atrás da carteirinha** (física ou digital)
- Pode ser solicitado por:
  - Dentista credenciado, OU
  - Próprio beneficiário

## Tipo de Guia

- Aceita **guia física em papel**
- Aceita **guia digital** (foto ou lançada no sistema)
