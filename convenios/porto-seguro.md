# CONVÊNIO — PORTO SEGURO ODONTOLÓGICO

## Exames Aceitos

O convênio Porto Seguro Odontológico aceita os seguintes exames na IX Radiologia:

- Documentação Ortodôntica  
- Panorâmica  
- Periapical / Interproximal / Bite Wing  
- Levantamento Periapical  
- Carpal  
- Radiografia de ATM  
- Telerradiografia  
- Moldagem  
- Fotos Intraorais e Extraorais  
- Modelo Ortodôntico  

## Exames Não Aceitos

- Tomografia Computadorizada — **não habilitado para este exame**
- Escaneamento Intraoral — **não é realizado por convênio**

## Guia

### Tipo de Guia
- Aceita **guia física** em papel  
- Aceita **guia digital** (foto ou lançada no sistema)

### Validade da Guia
- **30 dias** a partir da data de emissão

## Observações Importantes

- **Tomografia Computadorizada**:  
  Não estamos habilitados para realizar tomografia pelo convênio Porto Seguro Odontológico.

- **Porto Seguro Saúde (convênio médico)**:  
  Se o cliente mencionar “Porto Seguro Saúde”, informar que **não aceitamos convênio médico**, apenas o **Porto Seguro Odontológico**.

- **Quando o cliente disser que o dentista já lançou a guia ou que não precisa de guia**:  
  Coletar **nome completo, CPF e data de nascimento** e **transferir para atendente** para verificação no sistema.
